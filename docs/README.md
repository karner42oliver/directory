# Directory Plugin

*Mit dem Verzeichnis-Plugin kannst Du eine vollständige Verzeichnis-Webseite erstellen.**

## Directory fügt WordPress ein voll funktionsfähiges Verzeichnis mit Bewertungen, Rezensionen und Zahlungen hinzu.

Richte schnell ein Nischenverzeichnis oder sogar Deine eigene Webseite im Stil der Gelben Seiten ein.
![Klare, einfach zu navigierende Auflistung der Webseiten.](http://premium.wpmudev.org/wp-content/uploads/2010/11/directory-home-735x470.jpg)
Übersichtliche, einfach zu navigierende Auflistung der Webseiten.

### Mehr Möglichkeiten, Geld zu verdienen

Lade von Deinen Benutzern die Eintragung ihres Unternehmens in Dein exklusives Verzeichnis ein oder zertifiziere Unternehmensstandards und baue ein vertrauenswürdiges Netzwerk von Elite-Diensten auf, für deren Zugriff ein Abonnement erforderlich ist. Baue eine solide Nutzerbasis auf und nutze die enthaltene Banner-Werbefläche, um Einnahmen zu generieren. Und da Directory vollständig Multisite-kompatibel ist, kannst Du es mithilfe von PS Bloghosting als Upgrade in Dein Netzwerkdienstpaket integrieren.

### Leistungsstarke Navigation

Das Verzeichnis erleichtert die Verwaltung tausender Einträge. Erstelle benutzerdefinierte Benutzerberechtigungen und konfiguriere, wer Einträge bearbeiten, ändern, aktualisieren und moderieren kann. Ermögliche Benutzern das Hinzufügen von Einträgen über eine einfache Front-End-Oberfläche.
![Der Frontend-Editor erleichtert Benutzern das Hinzufügen und Bearbeiten von Einträgen.](http://premium.wpmudev.org/wp-content/uploads/2010/11/front-end-listing-1470x940.jpg)
Das Verzeichnis ist für maximale Effizienz strukturiert und nutzt die Plugin-spezifischen Kategorien, Tags und die Tabellensuche voll aus.
![Blitzschnelle Anpassungen.](http://premium.wpmudev.org/wp-content/uploads/2010/11/color-wheel-735x470.jpg)

### Passt zu jedem Thema

Directory wird mit einem eigenen klaren und einfachen Design geliefert – sodass es mit wenig bis gar keiner Konfiguration fantastisch aussieht. Farbräder erleichtern die schnelle Markenzuordnung und umschaltbare Stilsteuerungen ermöglichen eine ultraschnelle Anpassung.

Oder bette Directory irgendwo auf Deiner Webseite in ein beliebiges Theme ein. Nutze unsere umfassende Shortcode-Bibliothek für eine schnelle Integration durch Kopieren und Einfügen. Einfach genug für ein internes Unternehmensverzeichnis, leistungsstark genug für die Verwaltung eines Premium-Verzeichnisdienstes.
